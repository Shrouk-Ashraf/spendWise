# Spend Wise User Story

## Registration User Stories
1. As a new user, I want to sign up with email and password, so that I can create my account.
2. As a returning user, I want to log in with my credentials, so that I can access my data securely.
3. As a user, I want to reset my password, so that I can recover my account if I forget it.
4. As a user, I want secure authentication, so that my financial data is protected Using Fingerprint for example.

## Dashboard
1. As a user, I want to see my total balance, so that I know how much money I have.
2. As a user, I want to see a summary of income and expenses, so I understand my spending habits.
3. As a user, I want visual charts of income vs expenses, so I can track trends easily.

## Income Management
1. As a user, I want to add income entries (salary, gifts, etc.), so I can track money I receive.
2. As a user, I want to edit income entries, so I can correct mistakes.
3. As a user, I want to delete income entries, so my data stays accurate.
4. As a user, I want to categorize income (Salary, Bonus, Other), so I can track sources.
5. As a user, I want view monthly summaries of income, so I know how income changes over time.

## Expense Management
1. As a user, I want to add expense entries (food, transport, bills), so I can track spending.
2. As a user, I want to edit expense entries, so I can correct mistakes.
3. As a user, I want to delete expense entries, so I maintain accurate records.
4. As a user, I want to categorize expenses (Food, Transport, Utilities, Entertainment), so I can analyze patterns.
5. As a user, I want view monthly summaries of expenses, so I can see where my money goes.

## Budgeting
1. As a user, I want to set budgets per category, so I don’t overspend.
2. As a user, I want alerts when spending approaches budget, so I can adjust behavior.
3. As a user, I want charts comparing budget vs actual spending, so I can track my goals visually.

## Reports & Analytics
1. As a user, I want weekly/monthly/yearly reports, so I can analyze trends.
2. As a user, I want category breakdowns for both income and expenses, so I can identify areas to optimize.
3. As a user, I want net cash flow charts, so I can see if I’m saving or overspending.

## Notifications & Reminders
1. As a user, I want reminders to log income/expenses, so I stay up to date.
2. As a user, I want alerts for upcoming bills, so I avoid late payments
3. As a user, I want notifications when approaching budget limits, so I can control spending.
   
## Settings & Personalization
1. As a user, I want dark/light mode, so I can use the app comfortably.
2. As a user, I want language preferences, so I can use the app in my preferred language.
3. As a user, I want secure storage for sensitive data, so my financial info is safe.

## Optional – Stripe / Visa Sandbox Integration
1. As a user, I want to connect a test Visa/MasterCard via Stripe, so I can see card transactions automatically.
2. As a user, I want automatic categorization of card transactions, so I can see how money is spent.
3. As a user, I want balance updates from the test card, so I can simulate real-world tracking.
